🚦 Traffic Violation Logger

A smart web-based system to detect, record, and manage traffic violations in real time. It helps authorities efficiently track violations, store evidence, and send instant notifications to vehicle owners.

📌 Features
Capture and log traffic violations
Store violation details with timestamps
Automated SMS alerts to vehicle owners (Twilio integration)
User authentication system
Simple dashboard to view and manage records
🛠️ Tech Stack
Python (Flask)
SQLite / MySQL
HTML, CSS, JavaScript
Twilio API for SMS alerts
⚙️ How It Works
Violation is detected or entered manually
Data is stored in the database
System triggers SMS notification to the registered number
Admin can review all violation records in dashboard
🚀 Setup Instructions
git clone https://github.com/your-username/traffic-violation-logger.git
cd traffic-violation-logger
pip install -r requirements.txt
python app.py
